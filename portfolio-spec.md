# Projects

## VoltNet

VoltNet explores machine learning models for predicting battery performance in electrified powertrain systems.

Work includes:

- modeling battery behavior using voltage, current, and temperature telemetry
- exploring physics-informed machine learning methods
- developing anomaly detection pipelines
- investigating data-driven battery performance prediction

---

## HQML-BMS

Hybrid Quantum Machine Learning framework for predicting lithium-ion battery thermal runaway.

Work includes:

- implementing quantum feature maps such as Angle Embedding and ZZFeatureMap
- building anomaly detection models using variational quantum circuits
- combining quantum feature extraction with classical machine learning models
- achieving 96% prediction accuracy on simulated datasets
- exploring embedded deployment feasibility
